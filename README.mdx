## ðŸ¤– Enable Local AI (Ollama)

<div style={{background:'#fef9c3',border:'1px solid #fde68a',borderRadius:'0.7em',padding:'1em',margin:'1em 0'}}>
<b>Optional:</b> For AI-powered commentary and deeper insights, you can run a local Ollama model.<br/>
<b>Statistical analysis works out of the box without AI!</b>
</div>

**1. Download & Install Ollama**<br/>
<a href="https://ollama.ai/download" target="_blank"><img src="https://ollama.ai/public/ollama-logo.svg" alt="Ollama Logo" style={{height:'2em',verticalAlign:'middle'}}/> Download Ollama</a><br/>
Follow the instructions for your OS (Windows, Mac, Linux).

**2. Start Ollama**
```bash
ollama serve
```

**3. Create the CS2 Coach Model**
```bash
ollama create cs2-coach -f ./modelfiles/cs2-coach-enhanced.modelfile
```

**4. Configure your `.env`**
```env
DISABLE_AI=false
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=cs2-coach
```
